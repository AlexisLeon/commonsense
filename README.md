# Common Sense

## Requirements

- Node.js
- yarn
- MongoDB

## Installation

- Clone the project: `git clone https://github.com/AlexisLeon/common-sense.git`
- Install app dependencies: `cd app && yarn install`
- Install server dependencies: `cd server && yarn install`

## App
  - Start server: `yarn run dev`
  - Test: `yarn test`

## API
  - Start server: `yarn run dev`
  - Test: `yarn test`
